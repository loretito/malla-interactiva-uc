## Getting Started

### Prerequisites

- [Yarn](https://classic.yarnpkg.com/en/)

### Installation

1. Clone the repo

```bash
https://github.com/loretito/malla-interactiva-uc
```

2. Install packages

```bash
yarn
```

3. Run

```bash
yarn dev
```

4. Build for production

```bash
yarn build
```
